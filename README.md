## 使用说明

### 安装依赖

```bash
pip install -e .[render,export]
```

### 启动爬虫

```bash
python -m neteasespider.playlists
```

### 获取歌曲详情

```bash
python -m neteasespider
```

### 导出到 Excel

```bash
python -m neteasespider.export
```

### 渲染可视化图表

```bash
python -m neteasespider.render
```
